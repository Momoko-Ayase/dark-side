## 所以，具体为什么拖到现在我还是挂了？

我不知道你们对分手的定义是什么。但我至少觉得，分手之后，我没有任何理由去干除了善后事宜（特别是经济纠纷）以外的，在任何方面继续提供帮助。

但我还是做了。

提供Apple Account用于其更新日服/港服游戏，提供登录游戏和引继账户的帮助，甚至借了钱，因为其称“学校饭菜难吃而父母不给钱”。

哪怕就在前几天，还在问我如何登录其SEGA ID。

但我从中获得了什么呢？我给其提供的所有帮助都没有得到任何（哪怕是口头上的）回报（感谢）。

而且，她找我的时候我能尽快回复，我找她（基于对其请求的回应）永远别想及时得到响应。

我就举一个例子。一个多月前，我自己的代理服务因为不可抗力（Azure额度耗尽）需要更换节点重新搭建，很显然，此过程至少得更换配置文件，并且因为我更换了协议的原因客户端App也得换。她找我的时候我说明了这一点，并让其在空闲时间找我来帮助配置。结果呢？到现在都没有。（至于最近跑到Twitter上发了条推顺便偷摸摸把头像和简介改掉了，谁提供的帮助呢？好难猜啊。）

甚至前几天问我SEGA ID的事，还是我在那开着翻译软件在那翻了半天[找到了SEGA ID的创建规则](https://support-segaid.sega.com/hc/ja/articles/16284252186649)然后加了点猜测想出来告诉她的。
