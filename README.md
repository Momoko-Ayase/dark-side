## About / 这里为什么存在？

虽说这里叫“桃桃の阴暗面”，但是一般的负能内容不会出现在这里。

会出现在这里的，都是那些**没有给我任何尊重的人或事**，严重影响了我的个人生活的。

既然没有让我得到最起码的尊重，那么出现在这里也不会有任何问题。

如果这里出现了你，自己憋着。**你自己心里最清楚你干了些什么。**
